# 🧠 Second Brain

A personal knowledge management app that allows users to store and organize information from **YouTube**, **Twitter**, **web links**, and personal notes—all in one place.

---

## 📌 Features

- Save and organize:
  - 📺 YouTube videos
  - 🐦 Twitter posts
  - 🔗 Web links
  - 📝 Personal notes
- Share content with other users
- User authentication with JWT
- Tagging and filtering system
- Clean and responsive UI

---

## 🛠 Tech Stack

### Frontend
- ⚛️ **React**
- 📘 **TypeScript**
- 🌐 **Axios** for API requests
- Tailwindcss 

### Backend
- 🚀 **Node.js**
- 🧩 **Express**
- 🛡 **JWT** for authentication
- 🔗 **CORS** for cross-origin communication
- 📦 **TypeScript** for type safety

### Database
- 🍃 **MongoDB** with Mongoose ODM
